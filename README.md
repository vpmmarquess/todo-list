# To-Do List

Projeto de lista de tarefas desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar como concluída
- Filtrar tarefas
- Salvar dados com localStorage

## Tecnologias
- HTML
- CSS
- JavaScript
